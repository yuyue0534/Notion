### 一、Proton Mail 精简指南

核心亮点
- 瑞士隐私、全端到端加密
- 一套账号：Mail + Calendar + Drive + Docs + VPN + Pass
- 匿名注册，无需手机号

免费版核心能力
- 邮箱：500MB
- Drive：1GB
- VPN：每日 1GB
- 别名：10 个
- 加密邮件、日历、联系人

必用技巧
- 别名防骚扰
Settings → Addresses → Add alias
- 对外发加密邮件
写邮件 → 开启加密 → 设置密码
- 自毁邮件
写邮件 → 开启 Self-destruct
- 关闭已读追踪
Settings → Privacy → 关闭邮件追踪
- 2FA 两步验证
Settings → Account → Password & Security
- 免费 VPN
选日本 / 新加坡节点，日常够用

### 二、Tuta Mail 精简指南

核心亮点
- 德国 GDPR、开源、量子安全
- 免费版 = 完整邮箱 + 日历 + 联系人
- 无广告、极简、全内容加密

免费版核心能力
- 存储：1GB
- 别名：10 个
- 无 Drive / VPN / Docs
- 全端到端加密

必用技巧
- 别名
Settings → Aliases
- 对外加密发送
写邮件 → Encrypt → 设置密码
- 自毁邮件
写邮件 → Self-destruct
- 2FA
Settings → Security → 2FA
- 会话自动登出
Settings → Security → Session timeout
备份导出
Settings → Export
